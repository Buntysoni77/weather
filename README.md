# 🌦️ Smart Weather Predictor 

An interactive web application built with **Flask** and **Machine Learning** concepts. This project features a dynamic, real-time background that changes based on different weather conditions using the HTML5 Canvas API.

## 🚀 Live Features
- **Dynamic UI:** Real-time animations for Sunny, Rainy, Snowy, and Stormy weather.
- **Flask Backend:** Handles user input and processes weather logic.
- **Glassmorphism Design:** A modern, translucent interface for better user experience.

## 🛠️ Project Structure
- `app.py`: The main Flask server and routing logic.
- `static/`: Contains `style.css` and `weather.js` (Canvas animation).
- `templates/`: Contains `index.html` and `result.html`.
- `models/`: Folder for serialized Machine Learning models (.pkl files).
